# 版本規劃 v1.0.2

## 功能與更新說明

1. **動態讀取應用程式版本資訊 (Dynamic License Versioning)**
   - **目標**：在設定畫面中點擊「License」時，顯示與 `pubspec.yaml` 同步的正確版本號。
   - **實作內容**：
     - 引入 `package_info_plus` 套件。
     - 修改 `lib/settings/settings_screen.dart`，使用 `FutureBuilder<PackageInfo>` 搭配 `PackageInfo.fromPlatform()`。
     - 在 `showLicensePage` 中將原本寫死的靜態字串，改為動態帶入 `info.appName` 與 `info.version`。
     - 更新專案 `pubspec.yaml` 的版本號至 `1.0.2+1`。
